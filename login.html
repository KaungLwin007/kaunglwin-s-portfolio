ကျွန်ုပ်၏အကြောင်း

Hello! ကျနော့်နာမည်က ကောင်းလွင်ပါ Web Growing ကို အလွန် စိတ်ဝင်စားနေသော အထက်တန်းကျောင်းသားတစ်ယောက်ပါ။ လက်ရှိတွင်၊ ကျွန်ုပ်သည် code ရေးခြင်းနှင့် Web Design ဆိုင်ရာ ၏ တစ်စိတ်တစ်ပိုင်းအနေဖြင့် HTML ကို ကိုယ်တိုင်လေ့လာနေပါသည်။ ဤနယ်ပယ်တွင် ကျွန်ုပ်သည် အသစ်ဖြစ်သော်လည်း၊ Web Growing၏ အခြေခံများကို သင်ယူပြီး Websiteများကို ဖန်တီးပုံနှင့် ဒီဇိုင်းဆွဲနည်းကို ရှာဖွေခြင်းတို့တွင် စိတ်အားထက်သန်နေပါသည်။ ကျွန်ုပ်သည် လေ့လာသင်ယူနေသည့်အရာများကို အသုံးချကာ အတွေ့အကြုံများရရှိရန် ကျွန်ုပ်၏ကိုယ်ပိုင် websiteကို တည်ဆောက်ခြင်းကဲ့သို့သော ပရောဂျက်အသေးစားများကို လုပ်ဆောင်နေပါသည်။ ကျွန်ုပ်သည် အပြန်အလှန်အကျိုးသက်ရောက်သော Webpage များ ဖန်တီးခြင်း သို့မဟုတ် CSS အကြောင်း လေ့လာခြင်း ကို အထူးစိတ်ဝင်စားပါသည်။ ကျွန်ုပ်သည် ဤနယ်ပယ်တွင် ကျွန်ုပ်၏ အသိပညာနှင့် ကျွမ်းကျင်မှုများကို ချဲ့ထွင်ရန် စိတ်အားထက်သန်နေပြီး သင်ယူမှုအခွင့်အလမ်းအသစ်များကို စူးစမ်းရှာဖွေရန် စိတ်လှုပ်ရှားမိပါတယ်။ HTML မလေ့လာဖြစ်တဲ့အချိန်တွေမှာ ဂစ်တာတီးတာ၊ သီချင်းနားထောင်တို့လိုမျိုးလုပ်ရတာကြိုက်ပါတယ်။ စူးစမ်းချင်စိတ်နဲ့ စေတနာထားမှုက အောင်မြင်မှုရဲ့ သော့ချက်လို့ ယုံကြည်ပြီး ကျွန်တော်လုပ်တဲ့အရာတိုင်းမှာဒီစိတ်လေးထားပြီးလုပ်ဖြစ်ပါတယ်။ Instagram နှင့် Facebook တို့တွင် ကျွန်ုပ်နှင့် ဆက်သွယ်ရန် သို့မဟုတ် ကျွန်ုပ်၏ ပရောဂျက်များကို ဝင်ရောက်ကြည့်ရှုနိုင်ပါတယ်။ပိုမိုလေ့လာသင်ယူပြီး web growing ကမ္ဘာကို စူးစမ်းလေ့လာခြင်းအတွက် စိတ်လှုပ်ရှားမိပါတယ်။

ကျွန်ုပ်၏ လေ့လာမှုပန်းတိုင်များ

ကျွန်ုပ်သည် Web growing တွင် ကျွန်ုပ်၏ သင်ယူမှုပန်းတိုင်များစွာကို အောင်မြင်ရန် အာရုံစိုက်နေပါသည်။ ဒါကတော့ ကျွန်တော် လက်ရှိလုပ်ဆောင်နေတဲ့ အရာပါ။

HTML & CSS ကို ကျွမ်းကျင်စွာ တတ်မြောက်ခြင်း။

ကျွန်ုပ်၏လက်ရှိပန်းတိုင်မှာ ဖွဲ့စည်းပုံကောင်းမွန်ပြီး အမြင်အာရုံဆွဲဆောင်နိုင်သော web page များကို ဖန်တီးရန်အတွက် အကောင်းဆုံးအလေ့အကျင့်များအပါအဝင် HTML နှင့် CSS ကို ခိုင်မာစွာနားလည်ရန်ဖြစ်သည်။

အပြန်အလှန်အကျိုးပြုသော Website များ တည်ဆောက်ခြင်း။

ဖောင်များနှင့် ခလုတ်များကဲ့သို့ အပြန်အလှန်အကျိုးသက်ရောက်သော web element များ ဖန်တီးနည်းကို လေ့လာလိုပြီး အသုံးပြုသူအတွေ့အကြုံကို မြှင့်တင်ရန်အတွက် web page များတွင် ပေါင်းစည်းရန် စိတ်အားထက်သန်ပါသည်။

.

JavaScript အခြေခံများကို ရှာဖွေခြင်း။

HTML နှင့် CSS ကို အဆင်ပြေပြေ အသုံးပြုနိုင်သည်နှင့် တပြိုင်နက် ကျွန်ုပ်၏ Web project များတွင် တက်ကြွသော လုပ်ဆောင်နိုင်စွမ်းကို ထည့်သွင်းရန် JavaScript ကို စတင်လေ့လာရန် စီစဉ်ထားပါသည်။
